# Day 1 checkpoint cutscene audio

`impact-hit-cc0.wav` and the impact layers in `scuffle-cc0.wav` use material from **Thwack Sounds 1.0** by Jordan Irwin (AntumDeluge), released under CC0. Source: https://opengameart.org/content/thwack-sounds

The remaining runtime sounds were procedurally authored for this project by `tools/prepare_day1_camera_cutscene_assets.py` at 48 kHz mono. The downloaded archive and its original license are retained under `source/`.
