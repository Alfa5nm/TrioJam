# Day 2 aftermath audio provenance

The cloth, package, latch, arming-click, siren, explosion, tinnitus, panic and
low-rumble cues are project-authored 48 kHz assets already stored in the
repository. Day 2 reuses the same lossless masters used by the Day 1/Day 3
sequences and routes them through the existing SFX, Ambience, Music and UI
buses. No lossy retranscode was introduced for this pass.

The crowd source and its external license are documented separately in
`day2-rally-crowd-pixabay.md`.
